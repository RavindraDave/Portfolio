FinIQ Projects

FinIQ Suite caters to order management, best execution, compliance, suitability, documentation, valuation and post-trade life cycle requirements across mutliple asset classes of over 40 financial institutions.

I started my career in IT with FinIQ in mid 2010 and worked as tester for a year, which provided me oppurtunity to understand and learn different financial instruments including derivatives, structured products, swaps, funds, bonds, cash equity and cash FX.

After I started development in 2011, I worked on mutliple projects, listing few key projects below.

eFXConnect

Request for Stream based quotes and online execution with leading market-makers for spot, forward, swap, forward-forward, non-deliverable forward and precious metals.

Key Contributions:

  • Designed and developed API framework for best price discovery using WCF technology and FIX protocol with multiple market makers, which provides low latency price streaming and booking. Framework also allows to use internal pricing for various FX trades, using internal rates and swap points for various business scenarios.
  • Designed and developed a web socket based framework using Publish/Subscribe pattern for price streaming, real time notification and alerts. This framework allows to real time callbacks for any updates user has subscribed.

Mighty FX API

FX Limit order API for UOB for Limit Order integration with mobile banking applications. API allows to create Market Orders, Limit Orders and view historical rates for pairs.

Key Contributions:

  • Supervised a team of 5 engineers as Project lead for development of Limit Order execution module with end to end integration with core banking system to facilitate real time debit/credit transactions, in record time of 4 months for go-live.
  • Designed and developed a API to integrate with Might FX mobile application for Market and Limit Order booking and execution for retail customers.

BCFX

BCFX system is highly customised version of FinIQ FX system for one of the major bank in Thailand, which covers Retails, Trade Finance, Limits, GL and Treasury Modules.

Key Contributions:

  • Module lead for design and development of various Market Position blotters, which provides dealers accurate view on banks current positions for SPOT and Swaps, and allows them to transfer position for hedging in downstream system following rules to split position in banks local currency. Position transfer also includes auto transfer based Pair/Currency level threshold limits or based on Time interval.
  • Designed and developed a API framework to integrate with banks phone system, which provides system ability to auto populate customer portfolio information on dashboard for dealers, if customer calls from their registered number.
